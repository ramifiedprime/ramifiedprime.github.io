I am on git, where you can find any of my mathematical programming that seems useful enough to share.  Presently there isn't much of use, but hopefully more useful things will happen over time.

  • Redei Symbols -- Redei symbols are a class of very interesting trilinear symbols.  I wrote some magma to actually compute these and also to compute the minimally ramified extensions required for them.  For information on the symbols, consult Stevenhagen's paper (ArXiv:1806.06250) on the topic.
  • Conference Materials -- Whilst organising Y-RANT IV I became frustrated with the lack of automation for conference booklets and name tags.  What I produced isn't perfect, and if I ever have time then I'll fix it up, but it at least allows for writing a clean TeX document with easy linking between the schedule and titles/abstracts of speakers.
  • CubicCounts -- Code to count complex cubic fields, ordered by product of ramified primes.  This repository contains a C implementation of an algorithm essentially due to Belabas (A fast algorithm to compute cubic fields), which counts such fields ordered by discriminant, with relevant changes made to count by product of ramified primes.
